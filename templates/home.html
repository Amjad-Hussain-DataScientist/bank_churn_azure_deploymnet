🏦 About This Project

This application predicts whether a bank customer is likely to churn using an optimized CatBoost Classifier. The project demonstrates a complete end-to-end Machine Learning workflow including Data Ingestion, Data Validation, Feature Engineering, Data Transformation, Hyperparameter Tuning, Model Training, Flask Deployment, Docker Containerization, Azure Cloud Deployment and CI/CD with GitHub Actions.
